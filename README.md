payum-redsys
============

[![Build Status](https://travis-ci.org/crevillo/payum-redsys.png?branch=master)](https://travis-ci.org/crevillo/payum-redsys)

Provide integration of redsys/sermepa gateways to [Payum](https://github.com/Payum/Payum).
